## Project 01 — NovaTech Bank Transaction Tracker

![NovaTech Bank Transaction Tracker](01_Bank_Transaction_Tracker_Preview.PNG)

### Overview
Monthly bank transaction tracker for NovaTech Solutions 
to monitor income, expenses, and running balance for March 2025.

### Formulas Used
- `=E2+IF(C3="Income",D3,-D3)` — Running Balance
- `=SUMIF(C2:C11,"Income",D2:D11)` — Total Income
- `=SUMIF(C2:C11,"Expense",D2:D11)` — Total Expense

### Features
- Auto running balance calculation
- Income/Expense color coding
- Monthly financial summary
- Bar chart visualization

### Tools
Microsoft Excel 2021
